#  Chrome Extension Setup Guide

Follow the steps below to **test or use this Chrome extension locally**.

## Installation Steps

1. **Clone the Extension Repository**

   * Clone or download the extension folder to your local device.

2. **Open Chrome Extensions Page**

   * Open Google Chrome and go to:
     `chrome://extensions/`

3. **Enable Developer Mode**

   * Toggle **Developer mode** ON (top-right corner of the page).

4. **Load the Extension**

   * Click **“Load Unpacked”** (top-left corner).
   * Select the cloned extension folder from your device.

5. **Enable & Use**

   * Make sure the extension is **enabled**.
   * The extension is now ready to use 

---

##  Notes

* Any changes made to the extension code require clicking **Reload** on the extension card.
* This setup is ideal for **local testing and development**.

Happy building!
